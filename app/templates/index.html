About the Project

This Customer Churn Prediction System uses Machine Learning techniques to predict whether a customer is likely to leave a company's service. The prediction is based on customer tenure, billing information, contract type, payment method, paperless billing preferences, and senior citizen status.

The project uses a Random Forest Classifier, which is known for high accuracy and strong performance on structured datasets. Businesses can use this prediction system to identify potential churn customers and improve retention strategies.

How Prediction Works

  1. User enters customer details.
  2. Flask receives the input data.
  3. Input features are passed to the trained ML model.
  4. Random Forest analyzes customer behavior patterns.
  5. The model predicts churn probability.
  6. Prediction result and confidence score are displayed.
